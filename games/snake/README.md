# 🐍 贪吃蛇

经典贪吃蛇游戏，控制蛇吃到食物并尽量获得高分！

## 🎮 游戏玩法

- 使用 **方向键** 或 **WASD** 控制蛇的移动方向
- 吃到红色食物可以得分并延长蛇身
- 食物吃完后会自动在随机位置生成
- 速度会随着分数增加而变快
- 避免撞到墙壁或自身
- 按 **P** 键暂停游戏

## 📊 得分规则

- 基础速度时吃完一个食物得 **10 分**
- 速度越快，得分倍率越高
- 最高分会自动保存到本地

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

## 🏆 最高分

游戏会自动记录你的最高分，存储在浏览器本地。
